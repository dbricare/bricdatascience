Project Description

This project comes from a kaggle competition that tries to predict outcomes for animals based upon information collected at intake. All data comes from the Austin Animal Center.

The highest scoring (i.e., lowest log-loss) model uses a cross-validated gradient tree boosting classifier.

Features used include: age upon outcome, gender, animal type (dog or cat), named or unnamed, date and time of intake, purebred or mix.

Project Details

  • Full project analysis on GitHub

  • Tools used include Python pandas and scikit-learn libraries
