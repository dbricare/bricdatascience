Project Description

Prospective students who would like to maximize their potential earnings for a given tuition fee must not only consider the dollar amount, but also how likely they are to meet that goal.

Reported here are the closest matches for alumni earnings and current tuition as calculated by a nearest neighbor algorithm based on euclidean distance.

Project Details

  • Full project analysis on GitHub

  • Tools used include PostgreSQL, Python (sqlalchemy, pandas) libraries
  • Data is from US Dept. of Education College Scorecard and spans years from 1996 to 2015
